# 경북연구원 보고서 작성 완전 지침서 written By SW Yoon(uvgotmail@gdi.re.kr)

## 1. 기본 구조 및 체계

### 1.1 전체 구성 요소

- **겉표지** → **내지표지** → **연구진** → **요약(Summary)** → **목차** → **본문** → **참고문헌**
- 모든 장은 홀수 페이지에서 시작
- 간지(章紙) 사용으로 각 장 구분

### 1.2 페이지 구성

- A4 용지 기준
- 여백: 상 25mm, 하 20mm, 좌 25mm, 우 20mm
- 홀수 페이지 시작 원칙 (빈 페이지 삽입)
- 페이지 번호는 하단 중앙

## 3. 문체 및 표현 규칙

### 3.1 기본 문체

- **명사형 종결어미 사용**: "~임", "~함", "~할 것임"
- **객관적이고 간결한 문체**
- **학술적 정확성 유지**

### 3.2 표현 규칙

- 숫자는 한글과 아라비아 숫자 혼용
- 단위는 띄어쓰기: "26.3만 명", "9.9%"
- 괄호 사용: "대구시 발달장애인 : 12,228명(전체 장애인의 9.6%)"
- 참고 표시: "※ 참고표시"

## 4. 요약(Summary) 작성법

### 4.1 구조

```
Summary 요약                                                    i

1. 과제의 필요성
[연구 배경과 목적을 3-4문단으로 요약]

2. 현황과 문제점
[현재 상황 분석과 문제점을 구체적 수치와 함께 제시]

3. 정책 방향
[연구 결과와 정책 제언을 단기/중장기로 구분하여 제시]

```

### 4.2 내용 작성법

- 각 항목당 200-300자 내외
- 핵심 수치와 통계 포함
- 구체적이고 실행 가능한 방안 제시

## 6. 본문 작성 구조

### 6.1 장(章) 구성

각 장은 간지로 시작하며 다음과 같은 구조:

```
제1장
연구 개요
_________________________________

1. 국제회의산업의 중요성
2. 국제회의산업 육성정책
3. 국제회의산업의 중요성

```

### 6.2 절(節) 구성

```
제1절 연구의 필요성 및 목적 (필요시)

1. 세계가 주목하는 국제회의산업 (필요시)
1) 산업 (필요시)

■ 세계 MICE 시장 (필요시)
❍ 발달장애인은 지적 장애인과 자폐성 장애인을 통틀어 이르는 말로...
- 대구시 발달장애인 : 12,228명(전체 장애인의 9.6%)

```

### 6.3 위계 구조

1. **대항목 (■)**: 주요 주제
2. **중항목 (❍)**: 세부 설명(조금 긴 개조식으로 너무 짧지 않게 설명)
3. **소항목 (-)**: 부연 설명
4. **참고사항 (※)**: 주석이나 참고 정보

## 7. 표 작성 가이드

### 7.1 표 제목 형식

```
[표 1-1] 포항시 전체 제조업과 블루밸리 유치업종 간의 연평균 성장률 비교
(단위 : 명, 원)

```

### 7.2 표 구조

- 헤더는 음영 처리
- 숫자는 우측 정렬
- 단위 표기 필수
- 주석과 자료 출처 명시

### 7.3 표 예시

```
구분 | 연평균 성장률(%)
     | 사업체 수 | 종업원 수 | 출하액
포항시 전체 제조업(A) | 4.76 | -0.03 | 8.49
포항블루밸리 유치업종(B) | 6.27 | 0.03 | 9.18
연평균 성장률 차이(B-A) | 1.51 | 0.06 | 0.69

주 : 1. 종업원 수 100이상 기준
    2. 2012. 9. 30 기준
    3. 블루밸리란 .... 말한다 (대구경북연구원, 2011)
자료 : 통계청, 국가통계포털(http://kosis.kr), 2012. 9

```

## 9. 각 장별 내용 가이드

### 9.1 제1장: 연구 개요

- **연구 배경**: 문제 인식과 연구 동기
- **연구 목적**: 구체적 목표 설정
- **연구 내용**: 세부 연구 범위
- **연구 방법**: 방법론과 접근법
- **기대효과**: 활용 방안

### 9.2 제2장: 이론적 배경/현황 분석

- **관련 이론**: 이론적 틀
- **국내외 사례**: 벤치마킹
- **현황 분석**: 통계와 데이터 분석
- **문제점 도출**: 개선 필요사항

### 9.3 제3장: 실태 분석

- **조사 설계**: 조사 방법과 대상
- **조사 결과**: 데이터 분석
- **시사점**: 분석 결과의 의미

### 9.4 제4장: 정책 제언

- **기본 방향**: 정책 철학과 목표
- **세부 방안**: 구체적 실행 계획
- **추진 체계**: 거버넌스와 역할

### 9.5 제5장: 결론

- **연구 결과 요약**: 핵심 내용
- **정책적 시사점**: 실무적 함의
- **한계 및 향후 연구**: 연구의 제약

## 10. 참고문헌 작성법

### 10.1 국내 문헌

- **단행본**: 저자명, 발행연도, 『서명』, 발행지: 출판사
- **논문**: 저자명, 발행연도, 「논문명」, 『학술지명』, 권호
- **학위논문**: 저자명, 발행연도, 「논문명」, 학위구분, 대학명

### 10.2 외국 문헌

- **영문 표기**: 저자명, 연도, 서명(이탤릭), 출판사
- **웹사이트**: URL과 검색일자 명시

### 10.3 예시

```
정동영, 2002, 『가사문학연구』, 부산: 부산대학교 출판부
김기원, 1992, 「관여도에 따른 광고의 태도변인의 타당도 검증」, 석사학위논문, 계명대학교
http://kosis.kr/(통계청 국가통계포털)

```
